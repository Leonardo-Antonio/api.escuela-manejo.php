# Git Commands

```shell
$ git init // inicialiar un repo ( solo si es un nuevo proyecto<solo se hace 1 vez>)
$ git add . // agregar los cambios que realizaste
$ git commit -m "mesaje" // guarda los cambios (se crea el "historial")
$ git log // ver los commit y con ellos moverte entre ellos "viaje en el tiempo"
$ git brach // ver ramas
$ git brach <nombreRama> // crear una rama
$ git checkout <rama> // cambiar de rama
$ git checkout <IdCommit> moverte entre commits "moverte en el tiempo"
$ git merge <rama> // combina una rama de la otra (si no lo entindes luego te explico)
$ git push origin <rama> // subir tus cambio a tu repositorio remoto ( ya debes tener creado y enlasado tu proyecto local al remoto )
$ git pull // trae los cambios del repo remoto a tu local (sirve cuando se trabaja en equipo)
```

> Existem mas, pero estas son las que mas se usan
